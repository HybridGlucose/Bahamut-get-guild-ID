# 巴哈姆特抓ID私信整理
巴哈姆特私信抓帳號v0.2.2
本程式由HybridGlucose（a07051226@gmail.com，巴哈ID：aa07051226）撰寫  
特別寫給「巴哈姆特《歷史の紳士公會<ゝω・》」專用  
本程式用於抓取私信串帳號，目前功能接近完整  

##本程式具有以下功能：
* 可指定關鍵字
* 利用關鍵字抓該留言者帳號
* 自動產生私信用帳號格式
* 每20個帳號分行
* 顯示出私信人數
* 圖形化界面
 	
##本程式不具有以下功能，預計未來推出
* 最後尾的「,」須手動刪除
